================================================================================
DESIGN DECISIONS
================================================================================

1. Modular Architecture
   - Each component is independent
   - Easy to test and extend
   - Clean separation of concerns

2. Configuration-Driven
   - All behavior is configurable
   - No hardcoded paths or parameters
   - Easy to customize without code changes

3. Stateful Optimization
   - Complete state saved at checkpoints
   - Can resume seamlessly
   - Handles interruptions gracefully

4. Simulated LLM
   - Fast testing without API calls
   - Realistic behavior simulation
   - Easy to swap with real LLM

5. Comprehensive Logging
   - Every decision is logged
   - Supports both console and file output
   - JSONL for machine parsing

6. Greedy Strategy
   - Simple and interpretable
   - Fast convergence
   - Good for prompt optimization
